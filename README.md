## How To Install

**Type this command to your terminal/putty software:**

1) _`yum update -y`_
2) _`yum install -y git-core`_
3) _`git clone https://github.com/W3LLLL/OV6WHM.git`_
4) _`cd OV6WHM`_
5) _`chmod 777 install.sh`_
6) _`bash install.sh`_
